=======================
how to use 1-my_list.py
=======================

The module defines a class ``MyList(list)``

usage
=====

``MyList`` inherit from list a public instance method that prints a sorted list.

>>> mylist = __import__('1-my_list').MyList
>>> my_list = mylist()
>>> my_list.append(2)
>>> my_list.append(7)
>>> my_list.append(4)
>>> my_list.append(5)
>>> my_list.append(6)
>>> my_list
[2, 7, 4, 5, 6]
>>> my_list.print_sorted()
[2, 4, 5, 6, 7]

for negative integers
::

>>> MyList = __import__('1-my_list').MyList
>>> mylists = MyList()
>>> mylists.append(-3)
>>> mylists.append(-6)
>>> mylists.append(-2)
>>> mylists.append(-1)
>>> mylists.append(-9)
>>> mylists
[-3, -6, -2, -1, -9]
>>> mylists.print_sorted()
[-9, -6, -3, -2, -1]

::

>>> array = [2, 8, 5]
>>> my_list = MyList(array)
>>> my_list.print_sorted()
[2, 5, 8]

::
empty list check

>>> array = []
>>> my_list = MyList(array)
>>> my_list.print_sorted()
[]

::
same number check

>>> my_list2 = MyList()
>>> my_list2.append(3)
>>> my_list2.append(3)
>>> my_list2.append(3)
>>> my_list2.append(3)
>>> my_list2.print_sorted()
[3, 3, 3, 3]
